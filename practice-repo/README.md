# PRSchool Playground

这是 [PRSchool](https://prschool.dev) 的毕业练习仓库。

## 如何参与

1. Fork 这个仓库
2. 在 `students/` 目录下创建你的 JSON 文件
3. 提交 PR

## 文件格式

```json
{
  "name": "你的名字",
  "date": "2026-03-17",
  "learned": ["terminal", "git", "github", "PR", "claude-code"],
  "message": "一句话感想"
}
```

## 规则

- 只能在 `students/` 目录下添加/修改文件
- 文件必须是合法的 JSON
- PR 标题至少 5 个字符
- 通过自动审核后会自动合并

## 学员墙

每个成功合并的 PR 都代表一位 PRSchool 毕业生！🎓
